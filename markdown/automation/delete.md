# DELETE /io/board/:boardId/automation/:automationId
Delete a card automation.

### Example Request
```shell
curl -X DELETE \
  https://myaccount.leankit.com/io/board/1234/automation/5678 \
  -H 'Authorization: Basic base64encodedauthhere'
```

### Example Successful Response

204 No Content
