# DELETE /io/template/:templateId
Delete a board template.

### Example Request
```shell
curl -X DELETE \
  https://myaccount.leankit.com/io/template/1234234' \
  -H 'Authorization: Basic base64encodedauth' \
```

### Example Successful Response

204 No Content
