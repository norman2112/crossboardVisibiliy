# DELETE /io/board/:boardId/filter/:filterId
Delete a board filter.

### Example Request
```shell
curl -X DELETE \
  https://myaccount.leankit.com/io/board/943206946/filter/943867019 \
  -H 'Authorization: Basic base64encodedauthhere'
```

### Example Successful Response

204 No Content
