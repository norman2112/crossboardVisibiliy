# DELETE /io/card/:cardId/comment/:commentId
Delete a comment.

### Example Request
```shell
curl -X DELETE \
  https://myaccount.leankit.com/io/card/943206946/comment/943867019 \
  -H 'Authorization: Basic base64encodedauthhere'
```

### Example Successful Response

204 No Content
